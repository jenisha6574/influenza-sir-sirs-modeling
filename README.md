# Mathematical Modeling of Influenza Using SIR and SIRS Models

## Goal

The goal of this project was to understand how infectious diseases spread through a population and evaluate how different epidemiological factors influence outbreak dynamics. Using the SIR and SIRS compartmental models, this project examines the effects of infection rate, recovery rate, immunity loss, and vaccination on disease transmission.

## Objective

Develop mathematical models of influenza outbreaks using systems of ordinary differential equations and simulate disease progression under different parameter settings. The project investigates how changes in transmission, immunity, and vaccination strategies affect outbreak severity and long-term disease behavior.

## Methods

* SIR Model
* SIRS Model
* Ordinary Differential Equations (ODEs)
* Numerical Simulation
* Time Series Analysis
* Data Visualization in R

## Parameters Studied

* β (Infection Rate)
* γ (Recovery Rate)
* ω (Immunity Loss Rate)
* ν (Vaccination Rate)

## Key Findings

* Higher infection rates produced faster and larger outbreaks.
* Immunity loss led to persistent endemic infections.
* Vaccination significantly reduced infection peaks and outbreak duration.
* The SIRS model better captured diseases where immunity is temporary.
